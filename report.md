# ScreenInverter 项目进展报告

## 项目概述

屏幕颜色反转器 (ScreenInverter) 是一个 WPF 桌面应用，在屏幕任意区域上覆盖反色滤镜，缓解眼睛疲劳。特别适用于阅读不支持深色模式的 PDF 文档和网页。

### 3. 颜色反转（三种模式）

#### 模式 1: 智能文档模式（推荐）
**核心算法**：
1. 计算亮度 `luma = 0.299R + 0.587G + 0.114B`
2. 计算饱和度 `sat = max(R,G,B) - min(R,G,B)`
3. 判定逻辑：`(sat > 40) && (luma > 140)` → 保留原色（图片/高亮）
4. 否则强制去色 + 柔和亮度反转

**效果**：
- 白底 → 深灰 (#191919)
- 黑字 → 灰白 (#D7D7D7)
- ClearType 边缘 → 平滑灰色（消除彩色重影）
- 明亮彩色图片 → 保留原色，压暗 10%

#### 模式 2: 强力全反模式（柔和版）
**核心算法**：使用查找表对 R、G、B 每个通道独立柔和反转

**映射表**（`_softInvertTable`）：
| 输入 | 输出 |
|------|------|
| 0 (纯黑) | 215 (灰白) |
| 128 (中灰) | 120 (中灰) |
| 255 (纯白) | 25 (深灰) |

**效果**：色相翻转（红→青），亮度范围压缩，避免纯黑纯白

#### 模式 3: 仅反亮度模式（柔和版）
**核心算法**：
1. 计算亮度 `luma`
2. 查表获取目标亮度 `targetLuma = _softInvertTable[luma]`
3. 计算差值 `diff = targetLuma - luma`
4. 应用到 RGB 通道：`R' = R + diff`（保留色相）

**效果**：保留色相，仅反转亮度，同样避免纯黑纯白

## 已解决的问题

### 问题 1: 鼠标点击穿透 ✅ 已解决

**原问题**：点击窗口时，窗口会短暂闪烁，无法实现真正的点击穿透。

**解决方案**：
1. 使用 `SetWindowDisplayAffinity(WDA_EXCLUDEFROMCAPTURE)` 让窗口不被屏幕捕获"看见"
2. 移除 `Hide()/Show()` 流程，彻底消除闪烁
3. 使用 `WS_EX_TRANSPARENT` 样式实现真正的鼠标穿透
4. 添加锁定模式：按 `Ctrl+L` 切换穿透/交互状态

**技术要点**：
- `WDA_EXCLUDEFROMCAPTURE` (0x11) 需要 Windows 10 2004+
- 穿透模式下隐藏所有 UI 控件，只保留滤镜效果

### 问题 2: 屏幕捕获闪烁 ✅ 已解决

**原问题**：每次捕获时窗口会闪烁。

**解决方案**：使用 `SetWindowDisplayAffinity` 替代 `Hide()/Show()`，窗口在屏幕捕获时自动被"忽略"。

### 问题 3: 智能文档模式文字锯齿/重影 ✅ 已解决

**原问题**：
- 小字体边缘出现彩色光晕（红/青/紫重影）
- 笔画内部颜色不均匀，看起来像"被啃了一口"

**根本原因**：ClearType 抗锯齿与"保留色相"算法的冲突
- ClearType 使用高饱和度的红/蓝子像素平滑边缘
- 旧算法保留色相，把深红/深蓝边缘反转为粉红/淡青，形成彩色噪点

**解决方案**：
1. **提高图片判定门槛**：`(sat > 40) && (luma > 140)`，防止 ClearType 边缘被误判
2. **强制去色**：文字/背景模式下直接输出灰度，不保留色相
3. **添加 `BitmapScalingMode="HighQuality"`**：解决小字体像素缺失问题

**效果**：ClearType 彩色边缘 → 平滑灰色边缘 → 完美抗锯齿

### 问题 4: 强力全反/仅反亮度模式过于刺眼 ✅ 已解决

**原问题**：纯白反成纯黑、纯黑反成纯白，对比度过高

**解决方案**：使用柔和反转查找表 `_softInvertTable`
- 输出范围压缩到 `[25, 215]`
- 向中灰 (128) 靠近，减少视觉疲劳

## 核心算法说明

### 柔和反转查找表 (`_softInvertTable`)

```csharp
// 输入 [0, 255] → 输出 [215, 25]
output = 25 + (255 - input) * (215 - 25) / 255
```

**特性**：
- 避免纯黑 (0) 和纯白 (255) 输出
- 输出范围：`[25, 215]`
- 中灰 (128) 基本保持不变

### 智能文档模式判定逻辑

| 条件 | 判定结果 | 处理方式 |
|------|---------|---------|
| `sat > 40 && luma > 140` | 图片/高亮 | 保留原色，压暗 10% |
| 其他 | 文字/背景 | 强制去色 + 柔和亮度反转 |
